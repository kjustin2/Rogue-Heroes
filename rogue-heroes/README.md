# Rogue Heroes

Fast paced kingdom-building auto battler MVP.

## Browser

From this folder:

```powershell
npm run serve
```

Then open:

```text
http://127.0.0.1:8010/index.html
```

## Desktop App

Install dependencies once:

```powershell
npm install
```

Start the standalone Electron version:

```powershell
npm run desktop
```

## Checks

```powershell
npm run syntax
```

## Current Loop

Run your factory, spend gold on each owned base, optionally form a one-round alliance, use Battle Prep to order each base to attack an adjacent land or hold its offense home on defense, buy a spy report if you can afford it, watch your defense and offense resolve at the same time, choose a tech upgrade, then repeat until one kingdom controls the board.

Each kingdom starts with one base and neutral lands nearby. Attacks and sabotage only work across touching borders. Hover map castles to see land names and income. Use the map button to expand the kingdom board when you need a clearer view.
